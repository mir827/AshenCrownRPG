# 재의 왕관: 잊힌 서약

처음부터 브라우저 3D 게임으로 설계한 오리지널 판타지 RPG 챕터 1입니다. 외부 게임 에셋, 이미지, 음악, 원작 고유명사나 코드를 사용하지 않았습니다. 화면 속 캐릭터·건물·숲·성채·효과는 모두 Three.js 기본 도형과 직접 작성한 절차적 배치로 생성됩니다.

## 실행

Node.js 20 이상에서:

```bash
npm install
npm run dev
```

터미널에 표시되는 주소(기본 `http://localhost:5173`)를 브라우저에서 엽니다.

배포 빌드:

```bash
npm run build
npm run preview
```

## 조작

- `WASD`: 이동
- `Shift`: 달리기
- 마우스 우클릭 드래그: 3인칭 카메라 회전
- 휠: 카메라 줌
- `E`: NPC 대화/상자 조사
- `Space`: 기본 공격
- `Q`: 범위 스킬(6초 재사용)
- `R`: 회복약

## 실제 구현된 플레이 흐름

1. 국경 마을과 숲, 폐허, 점령 성채를 하나의 3D 공간에서 탐험
2. 촌장과 대화하여 정찰병 토벌 퀘스트 수주
3. 추적·공격하는 일반 적 3명과 실시간 전투
4. 봉인함에서 별철 검편을 얻고 레벨 2, 최대 HP와 공격력 상승
5. 몰락 기사단의 종기사 세라와 대화하고 동료 합류
6. 세라가 플레이어를 따라다니며 전투 지원
7. 관문 개방 후 일반 적보다 크고 강한 고대 병기 보스전
8. 보스 격파 시 챕터 결말과 다음 장 예고

HP/수치, 회복약, 스킬 쿨다운, 보스 HP, 대화창, 상호작용 안내, 퀘스트 로그 UI가 모두 실제 게임 상태와 연결됩니다.

## 데모 수준으로 단순화된 부분

- 캐릭터와 환경은 외부 에셋 없는 절차적 로우폴리 도형입니다.
- 충돌, 길 찾기, 전투 애니메이션은 챕터 데모에 맞춘 단순 계산입니다.
- 세이브, 장비창, 음성/음악, 복수 챕터는 포함하지 않습니다.
- 서사상 20~30분 분량의 첫 장 구조를 압축했으며 숙련도에 따라 실제 클리어는 약 8~15분입니다.
